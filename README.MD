# Proyecto base para la comision 999200928
## maquetacion proyecto web 
Se trata de maquetar en HTML y CSS un clon de la interfaz de Whatsapp , sin utilizar javascrip. Se incluyo una lista de chats a la izquierda y por la derecha tenemos el panel de conversaciones
## Estructura
El proyecto cuenta con los siguientas archivos y/o carpetas :
/assets (en esta carpeta se encuentran las imagenes utilizadas)
/styles/main.css (archivo de estilado CSS)
index.html (Archivo html pertinente al proyecto)
Readme.MD (el presente)
